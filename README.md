这是自用的web字体，字体相关的开源协议会在/credit文件夹放置。仓库中采用GPL协议。

因为其他某些字体的开源协议不兼容GPL，将放置在另一个仓库。

# USAGE
在css文件中加入
```css
@import url("https://gitee.com/sogeisetsu/myfont/raw/master/myfont.css");
```
